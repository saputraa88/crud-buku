CRUD Data Buku - Laravel

Project ini adalah aplikasi CRUD sederhana menggunakan Laravel, PHP, dan MySQL.

Fitur
- Tambah data buku
- Tampil data buku
- Edit data buku
- Hapus data buku

Teknologi
- Laravel
- PHP
- MySQL
- XAMPP

Cara Menjalankan
1. Clone repository
2. Jalankan `composer install`
3. Atur file `.env`
4. Buat database `crud_buku`
5. Jalankan `php artisan migrate`
6. Jalankan `php artisan serve`
